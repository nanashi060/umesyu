# Monetization

## RevenueCat

MVP includes state modeling only. Production setup must create iOS/Android apps, entitlement, offering, package, and env keys. Unconfigured state must show unavailable UI without crashing.

## AdMob

MVP shows no ads in development, screenshots, design-system, onboarding, or premium-disabled contexts. Production requires UMP/ATT/SKAdNetwork/App Privacy/Data Safety review.

## Amazon Associates

`EXPO_PUBLIC_AMAZON_ASSOCIATE_TAG` controls affiliate tagging. The app only opens an Amazon search handoff and does not compare prices, guarantee products, rank products, or handle checkout.
