# Product

## App

Umeshu Notes helps users prepare, age, and maintain homemade umeshu without creating an account.

## MVP Features

- Reminder plan for key umeshu dates: preparation day, first shake, monthly checks, tasting window, and bottling window.
- Tips library with bilingual preset content.
- Batch timeline as the +A feature: users can save ingredient ratios and track aging milestones.
- Optional photo URI for a batch.
- Share flow for a safe, non-sensitive summary.
- Amazon search handoff for supplies through an affiliate tag when configured.

## Audience

Home cooks and beginners who need a compact reminder and reference app during the seasonal plum window.
