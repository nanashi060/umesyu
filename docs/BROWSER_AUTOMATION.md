# Browser Automation

Use the Browser plugin / in-app browser for local Expo Web verification and deterministic screenshots.

Use the Browser plugin / in-app browser for `tools/app-store-screenshots` on localhost when checking editor layout, export behavior, or deterministic screenshots from the local Next.js editor.

Use existing Chrome only for logged-in account dashboards such as AdMob, Google Play Console, RevenueCat, PostHog, Sentry, Amazon Associates, or App Store Connect.

Do not commit browser profiles, cookies, tokens, auth screenshots, or account state.
