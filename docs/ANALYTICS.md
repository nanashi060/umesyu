# Analytics

## MVP State

Analytics is disabled unless configured. The adapter records no network events by default.

## Event Plan

Allowed event names:

- `onboarding_completed`
- `batch_created`
- `reminder_scheduled`
- `tip_opened`
- `share_started`
- `affiliate_search_opened`
- `paywall_viewed`
- `purchase_started`
- `purchase_completed`
- `ad_placeholder_viewed`
- `review_condition_met`

Do not send user-entered text, full notes, image URIs, or personal data.
