# Support

For Umeshu Notes support, open an issue on GitHub:

https://github.com/nanashi060/umesyu/issues

Please do not include private personal information, passwords, payment information, or sensitive account details in public issues.

The app does not require a login account. If you need help with local data, reminders, or photo permissions, include the device model, OS version, app version, and a short description of the behavior.
