# Privacy Policy

Effective date: June 14, 2026

Umeshu Notes is a local-first app for recording homemade umeshu batches, reminders, and tips.

## Data Collection

The app does not require an account and does not send batch notes, photos, reminder settings, or usage data to an external server in the MVP configuration.

Data is stored on the user's device, including:

- App settings
- Onboarding state
- Umeshu batch records
- Optional local photo URI selected by the user
- Reminder preferences
- Local review, purchase, and analytics state cache

## Photos

If the user chooses to attach a photo to a batch note, the app requests access to the photo library. The selected photo reference is stored locally on the device.

## Notifications

If the user allows notifications, the app schedules local reminders on the device. The app does not use push notification tokens or an external notification server.

## Third-Party Services

The MVP includes safe no-op adapters for purchases, ads, analytics, and error monitoring unless real configuration is provided. When these services are enabled in a future release, this policy and App Store privacy disclosures must be updated before release.

## External Links

The app may open an external browser when the user chooses a share or shopping handoff action. The app does not handle checkout or payment.
